## 📌 À propos du projet
Ce portfolio sert de vitrine professionnelle pour mon parcours étudiant. Il a pour objectif de :
* Exposer mes **compétences techniques** (Hard Skills) acquises en formation.
* Documenter mes **projets académiques** et personnels.
* Présenter mon parcours et mes motivations pour une recherche de **stage** ou d'alternance.

## 🛠️ Technologies utilisées
* **Langages :** HTML5, CSS3, JavaScript.
* **Outils :** Git & GitHub pour la gestion de version et le déploiement.
* **Hébergement :** GitHub Pages.

## 📂 Structure du Portfolio
Le site est organisé de manière à offrir une navigation claire pour un recruteur ou un enseignant :
1.  **Présentation :** Introduction sur mon profil et mes aspirations.
2.  **Compétences :** Liste détaillée des technologies maîtrisées.
3.  **Projets :** Galerie de mes réalisations avec descriptions et liens.
4.  **Contact :** Formulaire et liens vers mes réseaux professionnels.
