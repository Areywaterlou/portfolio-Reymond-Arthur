# Portfolio

Bienvenue sur mon portfolio.  
Ce projet présente mes compétences, mes projets et mon parcours en développement web.


## Technologies utilisées

- HTML5
- CSS3
- JavaScript (optionnel)
- Git & GitHub pour le versionnage

## Fonctionnalités

- Présentation personnelle
- Liste des compétences techniques
- Galerie ou section projets
- Formulaire de contact

## Installation et utilisation

1. Cloner le dépôt :

git clone https://github.com/Areywaterlou/portfolio.git

2. Ouvrir `index.html` dans un navigateur.
3. Modifier les fichiers pour mettre à jour les informations personnelles ou les projets.

## Contribution

Pour contribuer :

1. Forker le dépôt
2. Créer une branche pour vos modifications
3. Commiter vos changements
4. Faire un pull request
